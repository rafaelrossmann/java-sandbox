# 🧪 Java Sandbox

Repositório pessoal destinado a experimentações, exercícios acadêmicos e consolidação de fundamentos da linguagem Java e Orientação a Objetos.

## 📂 Estrutura do Repositório
- **`01-fundamentos/`**: Exercícios focados em lógica matemática e algoritmos.
- **`02-poo/`**: Práticas e exercícios de fixação sobre Programação Orientada a Objetos, encapsulamento, herança e polimorfismo.
- **`03-mini-projetos/`**: Pequenas aplicações criadas para resolver problemas lógicos isolados.

## 🛠️ Stack Utilizada
- Java (JDK 21+)
- IntelliJ Ultimate
- Git & GitHub
